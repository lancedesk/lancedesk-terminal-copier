# Security Policy

## Supported versions

Security fixes are provided for the latest release on `main`.

## Reporting a vulnerability

Please do not open public issues for suspected vulnerabilities.

Report privately by email to: `security@lancedesk.com`  
Subject line: `lancedesk-terminal-copier security report`

Include:

- clear description of the issue,
- affected version/commit,
- reproduction steps or proof-of-concept,
- impact assessment.

## Response targets

- Initial acknowledgement: within 3 business days.
- Triage decision: within 7 business days.
- Fix or mitigation timeline: communicated after triage.

## Disclosure process

- We validate and triage the report.
- We prepare a fix and release notes.
- We coordinate responsible disclosure timing with reporter when possible.
