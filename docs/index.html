LanceDesk Terminal Copier

Record terminal output. Copy complete logs instantly.

View on GitHub Releases

Why teams use ld

  • Capture full command + output history from real terminal sessions.
  • Copy logs with one command for bug reports, incidents, and AI prompts.
  • Keep a fast shell-native workflow without running a daemon.

Quick start

git clone https://github.com/lancedesk/lancedesk-terminal-copier.git
cd lancedesk-terminal-copier
chmod +x install.sh
./install.sh && source ~/.config/ld/ld.sh
